## C'est Quoi DWH?

### 🌟 Dirham World Halal

**DWH** est une crypto-monnaie Sharia-compliant sur **BNB Smart Chain**.

### Pourquoi DWH?

✅ **Sharia-Compliant** - Approuvé par conseil religieux
✅ **Transparent** - Blockchain public, auditable
✅ **Rapide** - Transactions en ~5 secondes
✅ **Économe** - Frais très bas (<$1)
✅ **Stable** - Adossé à actifs réels

## Configuration du Wallet

### Étape 1: Installer MetaMask

**Sur Desktop:**
1. Aller sur [metamask.io](https://metamask.io/)
2. Cliquer "Download"
3. Installer l'extension
4. Créer un wallet

**Sur Mobile:**
1. Télécharger l'app MetaMask (iOS/Android)
2. Créer un nouveau wallet
3. **SAUVEGARDEZ** votre seed phrase!

### Étape 2: Ajouter BNB Smart Chain

**Automatique:**
1. Ouvrir MetaMask
2. Cliquer sur le nom du réseau (en haut)
3. Cliquer "Add Network"
4. Chercher "BNB Smart Chain"
5. Cliquer "Add"

**Manuel:**
```
Network Name: BNB Smart Chain
RPC URL: https://bsc-dataseed1.binance.org:443
Chain ID: 56
Currency Symbol: BNB
Block Explorer: https://bscscan.com
```

### Étape 3: Ajouter le Token DWH

1. Ouvrir MetaMask
2. Sélectionner "BNB Smart Chain" (réseau)
3. Cliquer sur "Add Token"
4. Choisir "Custom Token"
5. Remplir:
   ```
   Token Contract Address: 0x559e10189ece533bf0c289961eeff1bb41755cb3
   Token Symbol: DWH
   Decimals: 18
   ```
6. Cliquer "Add Token"

## Acheter DWH

### Option 1: PancakeSwap (DEX)

**Meilleur pour**: Bas frais, liquidité élevée

1. Aller sur [pancakeswap.finance](https://pancakeswap.finance/)
2. Cliquer "Trade" → "Swap"
3. Sélectionner le réseau "BNB Smart Chain"
4. Entrer:
   - **From**: BUSD, USDT ou BNB
   - **To**: DWH
   - **Montant**: Votre montant
5. Cliquer "Swap"
6. Confirmer dans MetaMask

**Paires Disponibles:**
- DWH/BUSD
- DWH/USDT
- DWH/BNB

### Option 2: Biswap (DEX Alternatif)

1. Aller sur [biswap.org](https://biswap.org/)
2. Cliquer "Swap"
3. Même processus que PancakeSwap

### Option 3: Fonds

**Vous avez besoin de BNB d'abord:**

1. **Acheter BNB** sur:
   - Binance
   - Coinbase
   - Kraken
   - Autres exchanges

2. **Envoyer à MetaMask** (adresse BNB Smart Chain)

3. **Échanger BNB contre DWH** sur PancakeSwap

## Utiliser DWH

### Envoyer DWH

1. Ouvrir MetaMask
2. Sélectionner "DWH" token
3. Cliquer "Send"
4. Entrer:
   - **Adresse du destinataire** (0x...)
   - **Montant**
5. Cliquer "Next"
6. Vérifier les frais
7. Cliquer "Confirm"

**⚠️ Attention:**
- Vérifiez bien l'adresse 2 fois!
- Pas de retour possible si mauvaise adresse

### Recevoir DWH

1. Ouvrir MetaMask
2. Cliquer sur "Account Name" (en haut)
3. Cliquer "Copy Address"
4. Partager cette adresse avec l'envoyeur

**Votre adresse:**
```
0x... (commence par 0x)
```

### Vérifier le Solde

1. MetaMask: Voir directement dans wallet
2. [BSCScan](https://bscscan.com): Entrer votre adresse
3. [ApeSpace](https://apespace.io): Voir prix & stats

## Trading sur DEX

### Ajouter Liquidité (Optionnel)

**Sur PancakeSwap:**

1. Aller sur [pancakeswap.finance](https://pancakeswap.finance/)
2. Cliquer "Liquidity"
3. Cliquer "Add Liquidity"
4. Sélectionner paire (ex: DWH/BUSD)
5. Entrer montants
6. Cliquer "Supply"
7. Confirmer

**💰 Récompense:**
- Reçevez des frais de trading
- Risque d'impermanent loss

### Faire du Trading

1. Aller sur [pancakeswap.finance](https://pancakeswap.finance/)
2. Cliquer "Trade" → "Swap"
3. Entrer montants
4. Cliquer "Swap"
5. Confirmer transaction

## 🔐 Sécurité

### ✅ À Faire

- ✅ Sauvegarder votre seed phrase (écrit, pas numérique)
- ✅ Vérifier les adresses avant d'envoyer
- ✅ Utiliser des wallets officiels
- ✅ Vérifier les frais avant de confirmer

- ❌ Cliquer sur liens suspects

## 💰 Frais (Gas Fees)

- **Transaction simple**: $0.10-$0.50
- **DEX Swap**: $0.50-$2
- **Liquidity**: $1-$5

**Moins cher que Ethereum!**

### Réduire les Frais

1. **Attendre** : Frais ↓ quand réseau moins chargé
2. **Batching** : Faire plusieurs transactions ensemble
3. **Off-peak** : Trader la nuit/tôt le matin
