# Web Wallet - Dirham World Halal

## 🌐 Vue d'ensemble

Wallet web officiel pour DWH tokens.

**URL**: wallet.dirhamhalal.com (à venir Q2 2026)

## Fonctionnalités

### ✅ Actuelles

- Connect MetaMask
- View balance
- Send tokens
- Receive tokens
- Transaction history
- BSC explorer integration

### 🔄 Planifiées

- Multi-wallet support
- Trading interface
- Portfolio tracking
- Price charts
- Swap integration
- Mobile responsive

## Technologies

- Frontend: React.js
- Backend: Node.js
- Blockchain: Web3.js
- Wallet: MetaMask Web3Provider

## Sécurité

- ✅ No private keys stored
- ✅ HTTPS only
- ✅ Smart contract audited
- ✅ User funds safe

## Développement

```bash
git clone <repo>
cd web-wallet
npm install
npm start
```
