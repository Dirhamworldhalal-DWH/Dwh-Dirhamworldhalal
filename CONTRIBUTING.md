# Contributing to Dirham World Halal

## 🤝 Comment Contribuer

### 1. Fork le Repository

```bash
git clone https://github.com/Dirhamworldhalal-DWH/Dwh-Dirhamworldhalal.git
cd Dwh-Dirhamworldhalal
```

### 2. Create Feature Branch

```bash
git checkout -b feature/your-feature
```

### 3. Make Changes

Commit your improvements

### 4. Submit Pull Request

Describe votre contribution clairement

## 📝 Contribution Guidelines

### Documentation

- Clear & concise
- Examples inclus
- Liens pertinents
- Multilingue (FR/EN)

### Code

- OpenZeppelin standards
- Well commented
- Tested (unit & integration)
- Gas optimized

### Community

- Respectful & inclusive
- Sharia-compliant tone
- Constructive feedback
- Support others

## 🎯 Areas to Contribute

- 📝 Documentation
- 🐛 Bug fixes
- ✨ Features
- 🌍 Translations
- 📚 Educational content
- 🤝 Partnerships
