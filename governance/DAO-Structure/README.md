# DAO Governance - Dirham World Halal

## 🏛️ Structure

```
DWH Token Holders (Community)
         ↓
    DAO Smart Contract
         ↓
    Proposals & Voting
         ↓
    Governance Decisions
         ↓
    Sharia Board Review
         ↓
    Treasury Execution
```

## 🗳️ Voting Mechanism

### Eligibilité

- Détenteurs DWH tokens
- Minimum: 100 DWH (à confirmer)
- 1 token = 1 vote

### Processus

1. **Proposal Creation** - Tout détenteur
2. **Discussion** - 3 jours (communauté débat)
3. **Voting** - 5 jours (on-chain voting)
4. **Execution** - Si approuvé (>50%)

## 📋 Proposal Types

1. **Governance** - Rule changes
2. **Treasury** - Fund allocation
3. **Technical** - Smart contract upgrades
4. **Partnership** - New partnerships
5. **Sharia** - Islamic compliance

## 🔍 Oversight

**Sharia Board** supervise toutes décisions pour:
- ✅ Conformité Sharia
- ✅ Légalité
- ✅ Sustainability
